# るうまなの機能

## しるまな（Shiru-mana）— 知る
日本のマナー・文化・ルールを学ぶ掲示板です。
カテゴリ：交通マナー、公共施設、温泉・公衆浴場、食事・カフェ、買い物・支払い、宿泊・旅館、参拝・伝統、ゴミ・トイレ

各カテゴリには：
- コミュニティスレッド（質問・体験共有）
- るうまなガイド（公式マナーガイド）
- ルールとマナー集

## みるまな（Miru-mana）— 見る（Coming Soon）
エリア・防災マップ機能。現在地周辺の観光スポット、避難所、フリーWi-Fi、お手洗いなどを地図上で確認できます。

## きくまな（Kiku-mana）— 聞く（Coming Soon）
コミュニティ掲示板。体験を共有したり、質問したりできます。

## いくまな（Iku-mana）— 行く（Coming Soon）
文化体験・イベント予約機能。ワークショップ、お祭り、食体験、伝統文化体験を予約できます。

## おたすけ（Trouble）
緊急時のヘルプ機能。
- 警察（110番）、救急・消防（119番）への連絡ガイド
- 病気・けが、紛失・盗難、災害時の対処法
- 多言語電話相談窓口の情報

## 防災マップ（Coming Soon）
医療サービス、避難エリア、食料品店、警察・消防、公共施設を地図上で確認できます。

## その他の機能
- AI翻訳：スレッドやコンテンツをAIがリアルタイム翻訳
- ポイントシステム：活動に応じてポイントが貯まる
- プロフィール管理：表示名、アイコンの設定
- プッシュ通知：新着情報をお知らせ
