# るうまな（RUUMANA）について

## アプリの理念
るうまなは、日本に暮らす・訪れるすべての人のための「暮らしのパートナー」アプリです。
「知りたいマナーや解決策に、1タップでアクセス」をモットーに、日本の文化・マナー・生活情報を多言語で提供します。

## 名前の由来
「るうまな」= ルール + マナー を組み合わせた造語です。
日本語のひらがなで親しみやすさを表現しています。

## 対象ユーザー
- 日本に住む外国人（在住者・留学生・技能実習生）
- 日本を訪れる旅行者
- 日本の文化やマナーに興味がある方
- 日本で困っている方

## 対応言語
日本語（ja）、英語（en）、中国語簡体字（zh）の3言語に対応。
AI翻訳機能により、各種コンテンツをリアルタイムに翻訳できます。
