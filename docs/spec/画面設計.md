### 3.4 SCR-004: スレッド詳細

```
┌─────────────────────────────┐
│ [←] スレッドタイトル        │
├─────────────────────────────┤
│ パンくず: しるまな > 交通   │
│           > スレッドタイトル │
├─────────────────────────────┤
│ ┌────────────────────────┐  │
│ │ 👤 ユーザー名  1/15    │  │
│ │ 投稿本文テキスト...    │  │
│ │ [❤️ 3] [↩️ 返信]       │  │
│ │ [翻訳を表示]           │  │
│ └────────────────────────┘  │
│ ┌────────────────────────┐  │
│ │ 👤 ユーザー名2  1/16   │  │
│ │ ┌─引用──────────────┐  │  │
│ │ │ > 元の投稿テキスト │  │  │
│ │ └───────────────────┘  │  │
│ │ 返信テキスト...        │  │
│ │ [❤️ 1] [↩️ 返信]       │  │
│ └────────────────────────┘  │
├─────────────────────────────┤
│ [テキスト入力欄      ] [送信]│ ← 投稿フォーム
└─────────────────────────────┘
```

#### 操作仕様

| 要素 | 操作 | 動作 |
|------|------|------|
| ❤️ いいね | タップ | likes +1、localStorage で重複防止 |
| ↩️ 返信 | タップ | 返信先を引用して入力フォームにフォーカス |
| 翻訳を表示 | タップ | 投稿本文をAI翻訳（3層キャッシュ） |
| 引用部分 | タップ | 引用元の投稿までスクロール＆ハイライト（1.5秒） |
| 編集/削除 | タップ（投稿者本人のみ） | 投稿の編集/削除 |
| 送信 | タップ | 投稿を Firestore に保存 |

### 6.2 タイポグラフィ

| 要素 | サイズ | 太さ |
|------|--------|------|
| ページタイトル | `text-2xl` / `text-4xl` | Bold (700) |
| セクション見出し | `text-xl` | Bold (700) |
| 本文 | `text-sm` / `text-base` | Normal (400) |
| キャプション | `text-[9px]` / `text-[10px]` | Bold (600) |
| ボタン | `text-sm` | Bold (700) |

| アニメーション | 持続時間 | イージング |

| ステップ切替 | 250ms | ease |
| トースト | 300ms | ease |
| ボタンプレス | - | `active:scale-95` |
