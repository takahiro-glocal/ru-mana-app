## 1. プロジェクト概要

### 1.1 背景・目的

訪日外国人旅行者と地域の人々の間には、言語だけでなく文化やマナーの違いによる誤解や摩擦が生じることがある。本サービスは、異なる文化を持つ人同士が安心して交流できる社会を実現するための異文化理解プラットフォームである。

「知りたいマナーや解決策に、1タップでアクセス」をモットーに、日本の文化・マナー・生活情報を多言語で提供し、相互理解を促進する。

### 1.2 サービスコンセプト

| 原則 | 内容 |
|------|------|
| 否定しない | 文化の違いを優劣や正誤で判断しない |
| 争わない | 文化の違いで対立しない |
| 個を尊重する | 国籍や文化でカテゴライズせず一人ひとりを尊重 |
| 共生 | 相互理解と尊重を前提とした共生社会の実現 |
| 楽しく学ぶ | 文化を楽しみながら学べる環境の提供 |

### 1.3 名称の由来

「るうまな」は「ルール（Rule）」と「マナー（Manner）」を組み合わせた造語。堅苦しいものとしてではなく、文化として楽しく学べるサービスであることを表現。

## 2. 対象ユーザー

| ユーザー区分 | 説明 |
|-------------|------|
| 訪日外国人旅行者 | 日本の文化やマナーを理解し、安心して旅行や交流を楽しみたい人 |
| 国内店舗・企業・公共交通機関 | 外国人利用者とのコミュニケーションを円滑にしたい事業者 |
| 異文化交流に興味のある一般ユーザー | 世界の文化や習慣を楽しみながら学びたい人 |

## 4. 対応言語

| フェーズ | 言語 |
|---------|------|
| Phase 1（現在） | 日本語、英語、中国語（簡体字） |
| Phase 2（予定） | 韓国語、スペイン語、フランス語 |
| Phase 3（予定） | 訪日旅行者の多い国の言語へ順次拡張 |

---

## 5. 制約条件

- SPA（Single Page Application）として動作（SSR無効）
- Firebase Hosting による静的ファイル配信
- クライアントサイドで Gemini API を直接呼び出し
- 組織IAMポリシーにより Cloud Functions (Cloud Run) は使用不可
- API キーの HTTP リファラー制限による保護が必要
