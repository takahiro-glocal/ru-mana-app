# 非機能要件定義書

## ドキュメント情報

| 項目 | 内容 |
|------|------|
| プロジェクト名 | るうまな（RUUMANA） |
| バージョン | 1.0 |
| 最終更新日 | 2026-03-05 |

---

## 1. 性能要件

| 項目 | 要件 | 備考 |
|------|------|------|
| 初回ロード時間 | 3秒以内（LTE環境） | SPA初期バンドル |
| ページ遷移 | 500ms以内 | クライアントサイドルーティング |
| API応答（Gemini） | 10秒以内 | チャット・翻訳の生成 |
| Firestoreクエリ | 2秒以内 | リアルタイムリスナー初回取得 |
| 検索レスポンス | 300ms（デバウンス後即時） | クライアントサイドフィルタリング |

---

## 2. 可用性要件

| 項目 | 要件 |
|------|------|
| サービス稼働率 | 99.9%（Firebase Hosting SLA準拠） |
| 計画停止 | Firebase側メンテナンス時のみ |
| 障害復旧時間 | Firebase依存（自動復旧） |
| オフライン対応 | PWA + Firestore ローカルキャッシュにより一部閲覧可能 |

---

## 3. セキュリティ要件

| 項目 | 要件 | 実装方式 |
|------|------|----------|
| 認証 | Firebase Authentication | Google OAuth 2.0 / メール&パスワード |
| 認証永続化 | ブラウザローカルストレージ | `browserLocalPersistence` |
| 入力検証 | XSS防止 | DOMPurify によるサニタイズ |
| APIキー保護 | HTTPリファラー制限 + サーバーサイド保持 | Google Cloud Console / Cloud Functions 環境変数 |
| Firestoreルール | コレクション単位のアクセス制御 | Firebase Security Rules |
| 通信暗号化 | HTTPS必須 | Firebase Hosting 標準 |
| CORS | Google AI SDK のCORS対応 | generativelanguage.googleapis.com |

### 3.1 APIキー管理

| キー | 公開範囲 | 保護方式 |
|------|----------|----------|
| Firebase API Key | クライアント | Firebase Security Rules による制御 |
| Gemini API Key | サーバーサイドのみ | Cloud Functions 環境変数（クライアントに露出しない） |
| Google Maps API Key | クライアント | HTTPリファラー制限 + API制限 |
| OpenWeatherMap API Key | クライアント | ドメイン制限 |

---

## 4. 拡張性要件

| 項目 | 要件 |
|------|------|
| 言語追加 | i18n ファイル追加のみで対応可能な設計 |
| カテゴリ追加 | Firestore データ + i18n キー追加で対応 |
| 機能モジュール追加 | Nuxt pages + composables の追加で対応 |
| ユーザー数 | Firestore のスケーリングに依存（自動スケール） |

---

## 5. 運用・保守要件

| 項目 | 要件 |
|------|------|
| デプロイ方式 | `npm run deploy`（ビルド + サーバー依存関係インストール + Firebase Hosting & Cloud Functions デプロイ） |
| 環境変数管理 | `.env` ファイル（`env.dev` をテンプレート利用） |
| ログ | ブラウザコンソール + Firebase Analytics |
| モニタリング | Firebase Console ダッシュボード |
| バックアップ | Firestore 自動バックアップ（Firebase設定） |

## 7. ユーザビリティ要件

| 項目 | 要件 |
|------|------|
| レスポンシブデザイン | モバイルファースト設計 |

## 8. 国際化要件

| 項目 | 要件 |
|------|------|
| URL戦略 | `prefix_except_default`（`/` = 日本語、`/en` = 英語、`/zh` = 中国語） |
| ブラウザ言語検出 | 初回アクセス時にブラウザ言語を検出してリダイレクト |
| 言語Cookie | `i18n_redirected` で言語選択を保持 |
| 日付フォーマット | ロケール別の日付表示（`toLocaleDateString`） |
| AI応答言語 | ユーザーのロケールに応じた言語でAI応答 |

---

## 9. データ保持要件

| データ | 保持場所 | 保持期間 |
|--------|----------|----------|
| ユーザー情報 | Firebase Auth | 無期限 |
| スレッド・投稿 | Firestore | 無期限 |
| 翻訳キャッシュ | Firestore | 無期限 |
| ポイント履歴 | Firestore | 無期限 |
| フィードバック | Firestore | 無期限 |
| 閲覧履歴 | localStorage | ブラウザキャッシュクリアまで |
| いいね状態 | localStorage | ブラウザキャッシュクリアまで |
| 言語設定 | Cookie | ブラウザ設定に依存 |
