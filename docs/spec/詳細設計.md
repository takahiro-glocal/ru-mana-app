## 2. Firestoreモジュール（useFirestore）

### 2.1 投稿 CRUD

#### addPost

```
入力: threadId, threadTitle, { body, userId, userName, userIcon, replyTo? }
処理:
  1. posts サブコレクションに addDoc
  2. threads ドキュメントの postCount を increment(1)
  3. threads ドキュメントの updatedAt を serverTimestamp()
```

#### updatePost

```
入力: threadId, postId, body
処理: posts/{postId} の body フィールドを updateDoc
```

#### deletePost

```
入力: threadId, postId
処理:
  1. posts/{postId} を deleteDoc
  2. threads ドキュメントの postCount を increment(-1)
```

#### likePost

```
入力: threadId, postId
処理: posts/{postId} の likes を increment(1)
※ クライアント側で localStorage による重複チェック
```

### 2.2 リアルタイムサブスクリプション

#### subscribeToPosts(threadId)

```
クエリ: collection(db, 'threads', threadId, 'posts')
ソート: orderBy('createdAt', 'asc')
リスナー: onSnapshot で posts 配列をリアルタイム更新
戻り値: unsubscribe 関数
```

#### subscribeToCategoryThreads(categoryId)

```
クエリ: collection(db, 'threads')
  .where('categoryId', '==', categoryId)
  .orderBy('updatedAt', 'desc')
リスナー: onSnapshot で threads 配列をリアルタイム更新
```

#### subscribeToLatestThreads(limitCount)

```
クエリ: collection(db, 'threads')
  .orderBy('updatedAt', 'desc')
  .limit(limitCount)
リスナー: onSnapshot で threads + allThreads を更新
```

### 2.3 ポイントシステム

#### addPoints(userId, amount, reason)

```
処理:
  1. users/{userId} の points を increment(amount)
  2. users/{userId}/point_history に addDoc({amount, reason, createdAt})
```

#### getPointHistory(userId)

```
クエリ: users/{userId}/point_history
  .orderBy('createdAt', 'desc')
  .limit(20)
```

## 4. AI翻訳モジュール（useTranslation）

### 4.1 3層キャッシュフロー

```
translateText(id, text)
  │
  ├─ 1. メモリキャッシュ確認
  │    キー: `${id}:${locale}`
  │    → ヒット → return cached
  │
  ├─ 2. Firestoreキャッシュ確認
  │    コレクション: translations
  │    ドキュメントID: tr_{hashCode(text::locale)}
  │    → ヒット → メモリに保存 → return cached
  │
  └─ 3. Gemini API 呼び出し
       モデル: gemini-2.5-flash
       プロンプト: 文化的ニュアンスを保持した翻訳指示
       → 結果をメモリ + Firestore に保存 → return
```

### 4.2 ハッシュ関数

```typescript
const generateCacheId = (text: string, targetLang: string): string => {
  let hash = 0
  const str = `${text}::${targetLang}`
  for (let i = 0; i < str.length; i++) {
    const char = str.charCodeAt(i)
    hash = ((hash << 5) - hash) + char
    hash |= 0
  }
  return `tr_${Math.abs(hash).toString(36)}`
}
```
