### 1.2 デプロイフロー

```
[開発者]
  → npm run build (Nuxt 静的ビルド)

  → firebase deploy --only hosting
  → Firebase Hosting に配信
  → CDN経由でユーザーに配信
```

| `/kiku` | - | - | Coming Soonリダイレクト |

| デフォルト（ja） | 英語 | 中国語 |
|-----------------|------|--------|
| `/` | `/en` | `/zh` |

## 5. 状態管理設計

### 5.1 グローバル状態（useState）

| キー | 型 | composable | 用途 |
|------|------|-----------|------|
| `global-chat-open` | `boolean` | useChat | チャット開閉状態 |
| `global-chat-messages` | `ChatMessage[]` | useChat | チャット履歴 |
| `global-chat-typing` | `boolean` | useChat | タイピング表示 |
| `drawer-open` | `boolean` | useDrawer | ドロワー開閉状態 |
| `drawer-step` | `DrawerStep` | useDrawer | ドロワー現在ステップ |
| `ai-translations` | `Record<string, string>` | useTranslation | 翻訳メモリキャッシュ |
| `ai-translation-loading` | `Record<string, boolean>` | useTranslation | 翻訳ロード状態 |
| `user-history` | `BrowsingHistory[]` | useUserHistory | 閲覧履歴 |
| `gmaps_loaded` | `boolean` | useMapsLoader | Maps SDK読込状態 |

### 5.2 モジュールレベル状態（Ref）

| 変数 | composable | 用途 |
|------|-----------|------|
| `user` | useAuth | 認証ユーザーオブジェクト |
| `isAuthLoading` | useAuth | 認証初期化中フラグ |
| `categories` | useShiru | カテゴリ一覧キャッシュ |
| `allThreads` | useShiru | スレッド一覧キャッシュ |

### 5.3 ローカルストレージ

| キー | 用途 |
|------|------|
| `ru-mana-history` | 閲覧履歴（最大50件） |
| `ru-mana-notification-enabled` | 通知設定 |
| `liked-{threadId}-{postId}` | いいね状態 |

## 6. 外部API連携設計

### 6.1 Google Generative AI（Gemini）

| 項目 | 値 |
|------|------|
| SDK | `@google/generative-ai` |
| モデル | `gemini-2.5-flash` |
| 呼出元 | クライアントサイド（ブラウザ） |
| 用途 | AIチャット、AI翻訳 |
| 認証 | APIキー（HTTPリファラー制限推奨） |

### 6.2 Firebase

| サービス | SDK | 用途 |
|----------|-----|------|
| Authentication | `firebase/auth` | ユーザー認証 |
| Firestore | `firebase/firestore` | データベース（リアルタイム） |
| Storage | `firebase/storage` | アバター画像保存 |
| Hosting | Firebase CLI | 静的ファイル配信 |

### 6.3 Google Maps

| 項目 | 値 |
|------|------|
| SDK | `@googlemaps/js-api-loader` |
| ライブラリ | Places |
| 用途 | 防災マップ、ホームの地図表示 |

### 6.4 OpenWeatherMap

| 項目 | 値 |
|------|------|
| エンドポイント | `api.openweathermap.org/data/2.5/weather` |
| パラメータ | `lat`, `lng`, `units=metric`, `lang={locale}` |
| 用途 | ホーム画面の天気ウィジェット |

## 7. エラーハンドリング設計

| レイヤー | 方式 |
|----------|------|
| API呼出失敗 | try-catch + コンソールログ + ユーザーへのフォールバックメッセージ |
| 認証失敗 | Firebase エラーコード別のローカライズメッセージ |
| Firestore接続エラー | ローカルキャッシュからのフォールバック表示 |
| 翻訳失敗 | `null` を返し、原文を表示 |
| 位置情報取得失敗 | 東京（35.6895, 139.6917）にフォールバック |
| PWAインストール不可 | Coming Soonトースト表示 |
