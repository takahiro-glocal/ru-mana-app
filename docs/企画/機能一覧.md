# 機能一覧 — るうまな（RUUMANA）

---

## 1. メイン機能

### 1.1 しるまな（知る）— 公開中

| 機能 | 説明 | ユーザー価値 |
|------|------|-------------|
| カテゴリ別マナーガイド | 8カテゴリ（交通、公共施設、温泉、食事、買い物、宿泊、参拝、ゴミ） | 日本のマナーを体系的に学べる |
| OK/NG図解ガイド | 正しい例と間違った例を視覚的に表示 | 直感的にマナーを理解できる |
| Q&A掲示板 | ユーザーが質問・体験を投稿 | 実体験に基づくリアルな情報を得られる |
| コメント・返信 | スレッドへのコメント、返信引用機能 | コミュニティでの知識共有 |
| いいね機能 | 役立つ投稿への評価 | 質の高い情報が見つけやすい |
| 新規スレッド作成 | 自分で質問や情報を投稿 | 知りたいことを直接聞ける |
| キーワード検索 | スレッドタイトルで検索 | 必要な情報にすぐアクセス |
| リアルタイム更新 | 新しい投稿が即座に反映 | 常に最新の情報が見える |

### 1.2 みるまな（見る）— 構想中

| 機能 | 説明 | ユーザー価値 |
|------|------|-------------|
| 文化コンテンツ | 動画・記事・インフォグラフィック | 視覚的に文化を楽しく学べる |
| 文化クイズ | ゲーム形式で知識を確認 | 楽しみながら理解を深められる |

### 1.3 きくまな（聞く）— 構想中

| 機能 | 説明 | ユーザー価値 |
|------|------|-------------|
| 体験共有SNS | 文化体験や意見を共有 | 多様な視点から文化を知れる |
| コミュニティ交流 | 「理解」「共生」「尊重」ベースの交流 | 安心して交流できる場 |

### 1.4 いくまな（行く）— 構想中

| 機能 | 説明 | ユーザー価値 |
|------|------|-------------|
| イベント検索 | 文化体験イベントを探す | リアルな体験の機会を見つけられる |
| 予約機能 | ワークショップ・お祭り等を予約 | 簡単に文化体験に参加できる |

## 2. 緊急・安全機能

| 機能 | 説明 | ユーザー価値 |
|------|------|-------------|
| 緊急電話（110/119） | ワンタップで警察・救急に発信 | 緊急時に迷わず連絡できる |
| ステップガイド | 病気・紛失・災害の対処法を段階的に案内 | パニック時でも落ち着いて対応できる |
| 現在地コピー | 現在の住所をクリップボードにコピー | 通報時に正確な位置を伝えられる |
| 多言語相談窓口 | ひまわり、Visitor Hotline の案内 | 母国語で相談できる窓口を知れる |
| 防災マップ | 医療施設・避難所等を地図で表示 | 緊急時に最寄りの施設がわかる |

## 3. AI機能

| 機能 | 説明 | ユーザー価値 |
|------|------|-------------|
| まなアシスタント | AIチャットで文化・マナーの質問に回答 | いつでも気軽に質問できる |
| AI翻訳 | 投稿を日本語・英語・中国語に翻訳 | 言語の壁なくコンテンツを楽しめる |
| 文化的翻訳 | 単語の翻訳だけでなく文化的背景も自然に伝達 | 文化のニュアンスまで理解できる |
| 翻訳キャッシュ | 一度翻訳した内容を保存して高速表示 | 快適な翻訳体験 |

## 4. ユーザー管理機能

| 機能 | 説明 |
|------|------|
| Google認証 | Googleアカウントでワンクリックログイン |
| メール認証 | メールアドレスとパスワードで登録・ログイン |
| プロフィール編集 | 表示名・アバター画像の変更 |
| 自動アバター | プロフィール画像未設定時にAIアバターを自動生成 |
| ポイントシステム | 活動に応じたポイント付与と履歴表示 |
| 閲覧履歴 | 最近見たスレッドの履歴（最大50件） |
| フィードバック | アプリへの改善要望を送信 |
| 言語設定 | 3言語（日/英/中）から選択 |

## 5. プラットフォーム機能

| 機能 | 説明 |
|------|------|
| PWA | ホーム画面にインストールしてアプリとして利用 |
| レスポンシブ | スマートフォン・タブレット・PCに最適化 |
| 多言語URL | 言語別のURL（/en, /zh）でSEO対応 |
| ブラウザ言語検出 | 初回アクセス時にブラウザ言語を検出して自動リダイレクト |
| オフライン | ローカルキャッシュで一部機能がオフラインで動作 |
| 天気情報 | 現在地の天気をホーム画面に表示 |
| 今日のイベント | 日本の祝日・行事を自動表示 |

## 6. カテゴリ詳細（しるまな）

| カテゴリ | 学べる内容の例 |
|----------|---------------|
| 交通マナー | 電車・バスのマナー、優先席、スマホ通話、飲食ルール |
| 公共施設 | 静粛マナー、列の並び方、靴を脱ぐ場所 |
| 温泉・公衆浴場 | 入浴前の洗い方、タトゥーの扱い、水着禁止 |
| 食事・カフェ | いただきますの文化、箸のマナー、チップ、おしぼり |
| 買い物・支払い | 現金とカード、免税手続き、試着室マナー |
| 宿泊・旅館 | 室内での靴、浴衣の着方、チェックアウト |
| 参拝・伝統 | 神社参拝の作法、お辞儀、贈答文化、撮影ルール |
| ゴミ・トイレ | ゴミの分別、持ち帰りルール、リサイクル |
