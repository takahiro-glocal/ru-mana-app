## リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
| ユーザー獲得の困難 | 低MAU | SNSマーケティング、旅行メディア連携 |
| コミュニティの荒れ | ブランド毀損 | モデレーション機能、「否定しない」原則の浸透 |
| AI応答の品質 | ユーザー離脱 | ナレッジベースの継続的充実、フィードバック分析 |
| APIコスト増大 | 収支悪化 | キャッシュ最適化、従量課金の監視 |
| 競合サービスの出現 | シェア低下 | コミュニティと理念による差別化 |
