# サービス概要書 — るうまな（RUUMANA）

---

## サービス名

**るうまな（RUUMANA）**
「ルール」と「マナー」を組み合わせた造語

---

## ミッション

> 異なる文化を持つ人同士が安心して交流できる社会を実現する

---

## サービスコンセプト

「知りたいマナーや解決策に、1タップでアクセス。あなたの『知りたい』を支える、暮らしのパートナー。」

### 5つの原則

| 原則 | 意味 |
|------|------|
| 否定しない | 文化の違いを優劣や正誤で判断しない |
| 争わない | 文化の違いで対立しない |
| 個を尊重する | 国籍や文化ではなく、一人ひとりの「個」を見る |
| 共生 | 相互理解と尊重に基づく共生社会の実現 |
| 楽しく学ぶ | 堅苦しくない、楽しみながら学べる文化体験 |

---

## 解決する課題

### 訪日外国人側の課題
- 日本独特のマナーやルールがわからず不安
- 言語の壁で困った時に助けを求められない
- 文化の違いによる意図しないトラブル
- 緊急時（病気・紛失・災害）の対処がわからない

### 受け入れ側（店舗・企業・地域）の課題
- 外国人利用者とのコミュニケーションコスト
- マナー違反への対応に苦慮
- 多言語対応の負担

---

## サービス構成

### 4つの柱

| 機能 | 名称 | 概要 | 状態 |
|------|------|------|------|
| 知る | **しるまな** | 日本のマナー・文化を学ぶ掲示板＆ガイド | 公開中 |
| 見る | **みるまな** | 文化コンテンツで楽しく学ぶ（動画・クイズ等） | 構想中 |
| 聞く | **きくまな** | ユーザー同士の体験共有コミュニティ | 構想中 |
| 行く | **いくまな** | リアルな文化体験イベントの予約 | 構想中 |

### 補助機能

| 機能 | 概要 |
|------|------|
| おたすけ | 緊急時のヘルプ（警察・救急への発信、対処ガイド） |
| 防災マップ | 周辺の医療施設・避難所等を地図で確認 |
| まなアシスタント | AI搭載の文化コンシェルジュ |
| AI翻訳 | コンテンツのリアルタイム多言語翻訳 |

---

## 対象ユーザー

| セグメント | 詳細 | 想定規模 |
|-----------|------|----------|
| 訪日外国人旅行者 | 観光・ビジネス目的の短期滞在者 | 年間3,000万人以上 |
| 在日外国人 | 留学生・技能実習生・定住者 | 約300万人 |
| 国内事業者 | 外国人対応を行う店舗・企業・公共機関 | 宿泊・飲食・交通等 |
| 文化交流志向層 | 異文化に興味のある一般ユーザー | 潜在的に広い |

---

## 対応言語

| フェーズ | 対応言語 |
|---------|----------|
| Phase 1（現在） | 日本語、英語、中国語（簡体字） |
| Phase 2（計画） | 韓国語、スペイン語、フランス語 |
| Phase 3（構想） | 訪日旅行者の多い国の言語に順次拡張 |

---

## 技術的特長

| 特長 | 詳細 |
|------|------|
| PWA対応 | ホーム画面にインストールしてネイティブアプリのように利用可能 |
| AI搭載 | Google Gemini による高品質なチャット応答と翻訳 |
| リアルタイム | 掲示板の投稿がリアルタイムに反映 |
| オフライン対応 | ローカルキャッシュにより一部機能がオフラインでも利用可能 |
| サーバーレス | 自前サーバー不要でインフラコストを最小化 |

---

## 差別化ポイント

| 項目 | るうまな | 既存サービス |
|------|---------|-------------|
| コンセプト | 文化の相互理解・共生 | 情報提供のみ |
