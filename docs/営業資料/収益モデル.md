# 収益モデル — るうまな（RUUMANA）

---

## 1. 現在のコスト構造（Phase 1）

### 1.1 インフラコスト

| サービス | 月間コスト（目安） | 備考 |
|----------|-----------------|------|
| Firebase Hosting | 無料〜数百円 | 10GB/月まで無料 |
| Cloud Firestore | 無料〜数千円 | 日次50K読取・20K書込まで無料 |
| Firebase Auth | 無料 | 10,000 MAUまで無料 |
| Firebase Storage | 無料〜数百円 | 5GBまで無料 |
| Google Gemini API | 数千円〜 | リクエスト量に応じて従量課金 |
| Google Maps API | $200クレジット/月 | $200まで無料 |
| OpenWeatherMap | 無料 | 1,000コール/日まで無料 |
| ドメイン | 年額数千円 | カスタムドメイン使用時 |

### 1.2 初期段階の総コスト

**月額 0〜5,000円程度**（ユーザー数1,000人程度まで）

Firebase Spark（無料）プランで大部分がカバーされる。AI APIコストがスケールの主要因。

## 2. 収益化計画

### Phase 2-3: フリーミアムモデル

| プラン | 価格 | 内容 |
|--------|------|------|
| 無料プラン | ¥0 | 基本機能全て利用可能（広告表示あり） |
| プレミアムプラン | ¥500/月 | 広告非表示、AI優先応答、限定コンテンツ |

### Phase 3-4: 事業者向けサブスクリプション

| プラン | 価格 | 内容 |
|--------|------|------|
| スタータープラン | ¥5,000/月 | 多言語マナーガイドの施設埋込、基本分析 |
| ビジネスプラン | ¥20,000/月 | カスタムガイド作成、詳細分析、API利用 |
| エンタープライズ | 個別見積 | 自治体・大規模施設向けカスタマイズ |

### Phase 3-4: いくまなマーケットプレイス

| 収益源 | 料率 | 内容 |
|--------|------|------|
| 予約手数料 | 10-15% | いくまな経由のイベント予約に対する手数料 |
| プロモーション掲載 | 月額固定 | イベント主催者の優先表示 |

### Phase 4: 広告収入

| 広告タイプ | 単価目安 | 特徴 |
|-----------|----------|------|
| ネイティブ広告 | CPM ¥500-1,000 | コンテンツに溶け込む非侵入型広告 |
| スポンサードガイド | 月額 ¥30,000〜 | 事業者による公式ガイドのスポンサー |
